Explications
Acteurs :

Utilisateur : Représente l'utilisateur de l'application.
Cas d'utilisation :

Créer un compte (UC1) : L'utilisateur peut créer un compte.
Activer un compte (UC2) : L'utilisateur doit activer son compte via un courriel de vérification.
Se connecter (UC3) : L'utilisateur peut se connecter à l'application.
Accéder à la page d'accueil (UC4) : L'utilisateur peut accéder à la page d'accueil après s'être connecté.
Accéder à un réfrigérateur (UC5) : L'utilisateur peut accéder à la liste des réfrigérateurs et choisir un réfrigérateur spécifique.
Ajouter un aliment (UC6) : L'utilisateur peut ajouter des aliments dans un réfrigérateur.
Modifier un aliment (UC7) : L'utilisateur peut modifier les informations des aliments dans un réfrigérateur.
Supprimer un aliment (UC8) : L'utilisateur peut supprimer des aliments d'un réfrigérateur.
Recevoir alertes de péremption (UC9) : L'utilisateur reçoit des alertes de péremption pour les aliments.
Créer un réfrigérateur (UC10) : L'utilisateur peut créer un nouveau réfrigérateur (limité à 2 réfrigérateurs).
Modifier les informations de compte (UC11) : L'utilisateur peut modifier ses informations de compte.
Se déconnecter (UC12) : L'utilisateur peut se déconnecter de l'application.
Supprimer un réfrigérateur (UC13) : L'utilisateur peut supprimer un réfrigérateur et tous ses aliments.
Relations :

<<extends>> : Indique qu'un cas d'utilisation est une extension facultative d'un autre cas d'utilisation.
<<includes>> : Indique qu'un cas d'utilisation inclut obligatoirement un autre cas d'utilisation.
