@startuml
actor Utilisateur

' Enregistrement d'un utilisateur
Utilisateur -> PageEnregistrement : Accéder à la page d'enregistrement
PageEnregistrement -> Utilisateur : Formulaire d'enregistrement
Utilisateur -> PageEnregistrement : Soumettre le formulaire
PageEnregistrement -> Serveur : Enregistrer les informations
Serveur -> BaseDeDonnees : Enregistrer utilisateur
BaseDeDonnees -> Serveur : Confirmation d'enregistrement
Serveur -> EmailService : Envoyer courriel de vérification
EmailService -> Utilisateur : Courriel de vérification
Utilisateur -> Serveur : Clique sur lien de vérification
Serveur -> BaseDeDonnees : Activer le compte

' Connexion d'un utilisateur
Utilisateur -> PageConnexion : Accéder à la page de connexion
PageConnexion -> Utilisateur : Formulaire de connexion
Utilisateur -> PageConnexion : Soumettre le formulaire
PageConnexion -> Serveur : Vérifier les informations
Serveur -> BaseDeDonnees : Vérifier utilisateur
BaseDeDonnees -> Serveur : Utilisateur valide
Serveur -> PageAccueil : Afficher page d'accueil

' Activation de compte par courriel
Utilisateur -> EmailService : Clique sur lien de vérification
EmailService -> Serveur : Valider lien
Serveur -> BaseDeDonnees : Activer le compte
BaseDeDonnees -> Serveur : Confirmation activation
Serveur -> PageAccueil : Afficher page d'accueil

' Ajout d'un aliment dans un réfrigérateur
Utilisateur -> PageRefrigerateur : Accéder à la page du réfrigérateur
PageRefrigerateur -> Utilisateur : Afficher aliments actuels
Utilisateur -> PageRefrigerateur : Ajouter un aliment
PageRefrigerateur -> Serveur : Enregistrer aliment
Serveur -> BaseDeDonnees : Enregistrer nouvel aliment
BaseDeDonnees -> Serveur : Confirmation enregistrement
Serveur -> PageRefrigerateur : Mise à jour de la liste d'aliments

' Affichage des alertes de péremption d'aliments
Utilisateur -> PageAccueil : Accéder à la page d'accueil
PageAccueil -> Serveur : Demande des alertes
Serveur -> BaseDeDonnees : Récupérer alertes de péremption
BaseDeDonnees -> Serveur : Envoyer alertes
Serveur -> PageAccueil : Afficher alertes

@enduml
